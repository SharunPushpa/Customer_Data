# Customer Data Analysis

This project analyzes customer purchasing behavior using Python and Machine Learning techniques.

## Tools & Technologies
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

## Project Steps
- Data cleaning and preprocessing
- Exploratory Data Analysis (EDA)
- Data visualization
- Model building
- Model evaluation

## Files
- Customer_Data.ipynb – Full analysis with graphs and outputs
- Online Shop Customer Sales Data.csv – Dataset
